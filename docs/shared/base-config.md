# Base Configuration

## API Base URL
```
https://api.varansinagar.gov.in/api
```

## API Version
```
v1.0
```

## Host
- **Production**: api.varansinagar.gov.in
- **Staging**: staging-api.varansinagar.gov.in

## Authentication Method
JWT (JSON Web Token) - Obtained after successful OTP verification during login

## Rate Limiting
- **Requests per minute**: 60 per user
- **Daily limit**: 5000 requests per user

## Timezone
All timestamps are in IST (Indian Standard Time - UTC+5:30)
